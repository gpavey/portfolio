SF Film Nerd


WEBSITE

Im a bit of a film buff and wanted to know what movies were filmed in San Francisco. I found the data I needed on the SFGov data site and built a Rails app to take the data and present it on Google maps. A fun side project.


VISIT WEBSITE
THE PROBLEM

The data was not very clean to start out with. It had all the locations by name but not by geolocation so I had no easy way to map the data points. I also had to deal with the fact that multiple film were set at the same location.

THE SOLUTION

I found a tool from openrefine.org that would allow me to take the data, parse it up, send the named locations to google and get the geolocation codes back in a flat file. I was then able to take the new data and put it in a sql database. Fun stuff.
